1. **Data Cleaning**: Removed HTML tags, punctuation, stopwords, and applied lemmatization.
2. **Vectorization**: TF-IDF (top 5000 features).
3. **Modeling**: Logistic Regression
4. **Evaluation**:
   - Accuracy: **88.9%**
   - F1-score: **0.89**
   - Balanced class performance.
